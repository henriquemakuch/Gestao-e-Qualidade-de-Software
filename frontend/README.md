## Pré-requisitos

Antes de começar, certifique-se de ter o Node.js instalado.

```bash
npm install -g @quasar/cli
```

## Instalação

```bash
cd frontend
npm install
```

## Uso

Inicie o servidor de desenvolvimento:

```bash
quasar dev
```

Para compilar o projeto para produção:

```bash
quasar build
```

## Estrutura de Pastas

```
src/
|-- assets/
|-- components/
|-- layouts/
|-- pages/
|-- plugins/
|-- store/
|-- quasar.conf.js
```
